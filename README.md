# Proyecto-Web-Ejemplo
Proyecto web de ejemplo para aprender de Git y GitHub

## Descripción
Este proyecto contiene un sitio web simple con HTML y CSS, utilizado como ejemplo para prácticas de control de versiones con Git.

## Características
- Sitio web responsivo
- Estilos CSS personalizados
- Estructura HTML semántica

## Requisitos
- Navegador web moderno
- Git instalado en tu máquina local

## Instalación y Uso

### Clonar el proyecto
```bash
git clone https://github.com/brayansantiagocuta-eng/Proyecto-Web-Ejemplo.git
git checkout main
``` 

### Abrir el sitio
Abre `index.html` en tu navegador para ver el proyecto.
